⚡ Git 설치 후 EORA AI 배포하기

📥 Git 다운로드 및 설치

1 Git for Windows 다운로드

Git for Windows 다운로드
64-bit Git for Windows Setup을 다운로드하세요.
2 Git 설치

1. 다운로드된 Git-버전-64-bit.exe 파일 실행
2. 모든 옵션을 기본값으로 두고 "Next" 클릭
3. 설치 완료 후 PowerShell 재시작
3 Git 설정

PowerShell에서 다음 명령어를 실행하세요:
git config --global user.name "당신의이름"
git config --global user.email "your-email@example.com"

🔗 GitHub 저장소 생성

4 GitHub에서 새 저장소 생성

1. GitHub.com에서 새 저장소 생성
2. Repository name: eora-ai-complete
3. Description: EORA AI - 완전한 학습 및 회상 시스템
4. Public 또는 Private 선택
5. "Create repository" 클릭

💻 PowerShell 명령어로 배포

5 EORA 프로젝트 폴더로 이동
cd E:\eora_new
6 Git 저장소 초기화
git init
7 모든 파일 추가
git add .
8 첫 번째 커밋
git commit -m "EORA AI 완전한 학습 기능 구현 - 100% 작동 확인됨"
9 메인 브랜치 설정
git branch -M main
10 원격 저장소 연결
git remote add origin https://github.com/사용자명/eora-ai-complete.git
주의: "사용자명"을 실제 GitHub 사용자명으로 변경하세요!
11 GitHub에 푸시
git push -u origin main

🎉 배포 완료!

GitHub에서 https://github.com/사용자명/eora-ai-complete로 접속하여 확인하세요.

🔧 배포 자동화 스크립트

한 번에 실행하기

위의 명령어들을 하나씩 복사해서 실행하거나, 다음 배치 파일을 사용하세요:
deploy_complete_to_github.bat
