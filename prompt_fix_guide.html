🔧 프롬프트 관리 문제 해결

❌ 문제: 프롬프트 불러오기가 안 됨

프롬프트 관리자에서 ai_prompts.json 파일을 불러오지 못하는 문제가 발생했습니다.

✅ 해결 방법

stable_server.py가 수정되어 다음과 같이 작동합니다:

  • ✅ ai_prompts.json 파일을 여러 경로에서 검색
  • ✅ AI1의 system은 문자열로 처리
  • ✅ AI1의 role, guide, format은 배열로 처리
  • ✅ 프롬프트 관리 API 엔드포인트 구현 완료

📁 ai_prompts.json 파일 위치

서버는 다음 순서로 파일을 찾습니다:

1. ai_prompts.json (루트 디렉토리) 2. src/ai_prompts.json 3. src/templates/ai_prompts.json 4. src/ai_brain/ai_prompts.json

🔍 AI1 프롬프트 구조

ai_prompts.json에서 AI1의 구조:

{ "ai1": { "system": "문자열 형태의 시스템 프롬프트", "role": ["역할1", "역할2", "역할3"], "guide": ["가이드1", "가이드2"], "format": ["포맷1", "포맷2"] } }

🚀 서버 실행 방법

# 방법 1: Python 직접 실행 python stable_server.py # 방법 2: 배치 파일 실행 start_stable_server.bat

🌐 접속 주소

서버가 실행되면 다음 주소로 접속하세요:

  • 홈페이지: http://127.0.0.1:8100
  • 로그인: http://127.0.0.1:8100/login
  • 프롬프트 관리: http://127.0.0.1:8100/prompt_management
