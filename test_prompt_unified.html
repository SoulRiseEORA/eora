📋 개선된 프롬프트 관리 시스템

✅ 카테고리별로 모든 내용을 하나의 텍스트 영역에 표시

✅ 줄바꿈으로 구분된 항목들을 한눈에 편집 가능

✅ AI1의 system은 문자열로, 나머지는 배열로 자동 처리

✅ 더 직관적이고 편리한 인터페이스

System 프롬프트 예시

AI의 기본 시스템 설정과 동작 원칙을 정의합니다.

Role 프롬프트 예시

AI가 수행할 역할과 책임을 명시합니다.

💡 사용 방법

  1. 각 카테고리의 텍스트 영역에서 직접 편집
  2. 줄바꿈으로 항목 구분 (자동으로 배열로 변환)
  3. AI1의 system은 하나의 긴 텍스트로 작성
  4. 저장 버튼 클릭으로 변경사항 저장
