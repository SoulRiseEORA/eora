📝 4단계: 각 업로드별 상세 과정

4-1. 파일 선택
• "choose your files" 클릭 또는 드래그 앤 드롭
• Ctrl + A로 여러 파일 동시 선택 가능
• 폴더 전체를 드래그하면 하위 파일들도 자동 포함
• 최대 100개 파일까지 한 번에 업로드 가능
4-2. 업로드 확인
• 선택된 파일 목록 확인
• 파일 크기가 25MB 미만인지 확인
• 불필요한 파일이 포함되지 않았는지 확인
4-3. 커밋 메시지 작성
• 페이지 하단 "Commit changes" 섹션에서
• "Commit directly to the main branch" 선택
• 위의 예시 커밋 메시지 중 해당하는 것 사용
4-4. 업로드 실행
• "Commit new files" 버튼 클릭
• 업로드 진행 상황 확인 (시간 소요될 수 있음)
• 완료되면 저장소 메인 페이지로 리다이렉트

• 또는 "Add file" → "Upload files"
• 위의 2차, 3차, 4차 업로드 순서대로 반복
⚠️ 업로드 시 주의사항:
• 한 번에 너무 많은 파일을 업로드하지 마세요 (브라우저 부하)

• .env 파일은 절대 업로드하지 마세요
